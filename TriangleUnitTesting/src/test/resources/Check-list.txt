Чек-лист Triangle.


Тесты для метода checkTriangle():
1.	С тремя сторонами больше и не равными 0;
2.	С одной стороной равной 0 (3 шт.);
3.	С двумя сторонами равными 0 (3 шт.);
4.	Со всеми сторонами, равными 0 (1 шт.);
5.	С одной стороной  < 0 (3 шт.);
6.	С двумя сторонами < 0 (3 шт.);
7.	Со всеми сторонами < 0 (1 шт.);
8.	Сумма сторон a и b меньше c (a+b <=c);
9.	Сумма сторон a и с меньше b (a+c <=b);
10.	Сумма сторон b и с меньше a (b+c <=a);
11.	Стороны a, b, c  > 0 и сумма двух сторон больше чем третья сторона (a+b >c and a+c >b and b+c >a);
12.	С одной стороной равной нулю и одной стороной меньше нуля (3 шт.);
13.	С одной стороной равной нулю и двумя сторонами меньше нуля (3 шт.);
14.	С двумя сторонами равными нулю и одной стороной меньше нуля (3 шт.);

2.	С одной стороной равной 0 (3 шт.)( a<=0, b<=0, c<=0)
3.	С одной стороной меньше 0 (3 шт.)( a<=0, b<=0, c<=0)
4.	С суммой двух сторон меньше чем третья сторона(3 шт.)( a+b <=c, a+c <=b, b+c <=a)
5.	С тремя сторонами равными 0.
6.	С тремя сторонами меньше 0.

Тесты для метода detectTriangle():
1.	Со сторонами при которых выполняются условия a*a + b*b = c*c и a != b != c
2.	Со сторонами при которых выполняются условия b*b + c*c = a*a и a != b != c 
3.	Со сторонами при которых выполняются условия a*a + c*c = b*b и a != b != c  
4.	С одной или несколькими сторонами равными Double.MAX_VALUE(Exception).
5.	Со сторонами при которых выполняются условия a*a + c*c = b*b и a = b, (также a = c, b = c)
6.	Со сторонами при которых выполняются условия a = b = c
7.	Со сторонами при которых выполняются условия a=b!=c (также a=c!=b, b=c!=a)
8.	С одной стороной равной 0 (3 шт.) (Exception).
9.	С одной стороной меньше 0 (3 шт.) (Exception).
10.	С тремя сторонами равными 0 (Exception).
11.	С тремя сторонами меньше 0 (Exception).
